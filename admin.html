Discord paste

Strategy

Paste Discord #alerts messages (F##: s 6940 / F##: +12.5) or a Discord JSON export. Points-based.Paste Discord #options alerts in the multi-line ID: n / Ticker / Type / Strike / Expiry / Entry / Stop price / Default: -N points / ID n: +$$$ format. Dollar-based P&L. Both raw paste and DiscordChatExporter HTML are accepted.
